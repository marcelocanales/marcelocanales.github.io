En Mac OSx cuando se inicia Emacs desde el GUI, emacs no tomará las varibles de entorno definidas en el sistema a travez de los archivos .bashrc, .profile, .zshrc o cualquier otro, si no que tomará las variables de entorno de MacOSX plist, entonces dependiendo desde donde ejecutamos emacs, (terminal o gui) seran las variables de entorno que tomará emacs, para ver las variables de entorno: ;; show env var named path (getenv "PATH")
